💻 3. Tokens de Diseño (CSS Variables)

Copia y pega este bloque en tu archivo styles.css o :root para mantener la fidelidad de marca en el código:

:root {
  /* Colors */
  --color-primary: #e7472c;
  --color-secondary: #386bb7;
  --color-bg-light: #a5abaf;
  --color-bg-dark: #605d5c;
  --color-text: #000000;
  --color-white: #ffffff;

  /* Typography */
  --font-display: 'Zen Dots', sans-serif;

🕹️ 5. Componentes de Interfaz (UI Guide)

Botones (Buttons)

Primario: Fondo --color-primary, texto --color-white. Uso para acciones positivas (Enviar, Cotizar).

Secundario: Fondo transparente, borde --color-bg-dark, texto --color-text.

Estado Hover: Oscurecer el color de fondo un 10%.

Banners y Contenedores

Se recomienda el uso de Gris Plata para diferenciar secciones de servicios.

Uso de Gris Carbón para el footer y secciones de contacto técnico.

♿ 6. Guía de Accesibilidad (WCAG 2.1)

Para garantizar que el sitio web sea inclusivo:

Contraste: No usar el Naranja Ingedaca sobre fondos amarillos o rojos. Siempre sobre Blanco o Gris Plata.

Legibilidad: El texto en Aptos Light no debe ser inferior a 16px para el cuerpo del mensaje.

Interactividad: Todos los elementos clickeables deben tener un área mínima de 44x44px en dispositivos táctiles.
